A summary of the things I do

Here's a overview of my job description

Design and Implemention of Gameplay Mechanics

I design and implement interactive gameplay features and mechanics in collaboration with game designers. I write codes, scripts, and algorithms to create engaging and enjoyable gameplay experience.

Bug Fixing and Optimization

I troubleshoot and fix issues related to gameplay functionality, performance, and stability. I debug code, optimize systems, and ensure smooth gameplay experience by addressing bottlenecks, profilling, and optimizing code.

Collaboration and Iterative Development

I work closely with other team members like artists, level designers, and QA testers, to integrate and refine gameplay features. I participate in agile development processes, adapt to design changes, and iterate on gameplay mechanics based on user feedback and playtesting results.

I also play a lot of games and try to figure out the logic behind them; an attempt to see if it is what I can do or learn how to do.

SPLATTER BALL

Splatter ball is a 10 levelled, top down 2D, casual mobile android game built with 3D assets.

Els Where?

Global game jam 2023; worked in a complete game development team with each individual having a specific duty towards building this 3D adventure puzzle game on PC. I wrote the logic and code for the companion AI for the main character, I also collaborated with other programmers in the team to successfully implement story and gameplay features designed by the story tellers and 3D artists in the team to develop a playable build under 3 days.

Multiplayer game simulation

This is a complete multiplayer 2D PC game developed in Unity game engine using Photon Unity Networking to sync player movement and states.

Anna X20

Led a team of 5 to develop this exciting PC game with amazing UI and music in less than two weeks. I was the level designer, gameplay programmer, as well as the project manager for this game which was built during Level99 Game jam, 2023. Worked closely with a fellow gameplay programmer, a UI artist, an environmental artist, and a story teller in the development of Anna X20.

INTO THE VOID

Worked in a team of 4 for the Global game jam 2022, where we developed this 2D PC game with keyboard controls in less than 72 hours.

8BallPool Simulation

The playable build contains the self-coded logic of the gameplay of features of the popular mobile game 8BallPool. The multiplayer aspect and complete UI flow are still in development.

Simple Calculator

A calculator android app made with Unity game engine. I implemented a calculator API, designed the UI placements and wrote the individual button's logic.

Simple Animated 3D Hypercasual Game Implementation

The android game build simply demonstrates how a player follows a particular path, a very useful tool in building companion AI that shows a player the path to follow in an open world game.

AFRO REVERSE

My first ever collaborative game, later renamed to AfroLibra, is available on Google PlayStore. Building it was part of the requirements for graduation from the 6 months game development bootcamp conducted by GameUp Africa, Maliyo, the first of its kind. I worked collaboratively, and as a level designer, with co-learners to successfully build and upload our first ever game to Google PlayStore.
